Web Development Glossary

Repository
A central location on a service like GitHub where project files and version history are stored.
Commit
A saved snapshot of changes made to a file or set of files in Git.
Branch
A parallel version of a repository used for safely developing features without breaking the main code.
Tag
A keyword enclosed in angle brackets (like <p>) used to define elements in HTML.
